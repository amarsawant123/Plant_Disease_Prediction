# Plant_Disease_Prediction

The Plant Disease Prediction System is a Machine Learning-based project designed to identify and classify plant diseases from leaf images. The main objective of this project is to support smart agriculture by enabling early disease detection, reducing crop loss, and improving farming productivity.

This system uses image classification techniques along with data preprocessing, Exploratory Data Analysis (EDA), and visualization tools such as Matplotlib and Seaborn to analyze plant leaf datasets and improve prediction accuracy. The project was developed using Python in Jupyter Notebook and includes model training, feature engineering, and performance evaluation.

The model can classify healthy and diseased plant leaves based on image patterns and features, helping farmers and agricultural experts make faster and more accurate decisions.
